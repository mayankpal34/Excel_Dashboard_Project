Bike Buyers Dashboard Project (Excel)

Project Overview
This project is an interactive Bike Buyers Dashboard created in Microsoft Excel. It analyzes customer data to understand the factors that influence bike purchases.

 Features
- Interactive Dashboard
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Analysis

 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

 Dataset
The dataset contains:
- Age
- Gender
- Income
- Education
- Occupation
- Marital Status
- Region
- Commute Distance
- Purchased Bike

 Key Insights
- Bike purchases by age group
- Bike purchases by gender
- Income vs. bike purchase
- Commute distance analysis
- Regional comparison

 Files
- Bike_Buyers_Dashboard.xlsx
